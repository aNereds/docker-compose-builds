# docker-compose-builds

Docker compose self education 

## Installation & Usage

From cloned dir run:

```bash
docker-compose up --build - For Prod Mode Run

OR

docker-compose -f docker-compose.yml -f docker-compose.development.yml up --build - For Dev env/mode run
```
